## Installation

### From [Crates.io][crates.io]

```bash
cargo install repofetch
```

### From [this repo](https://github.com/spenserblack/repofetch)

```bash
git clone https://github.com/spenserblack/repofetch.git
cargo install --path ./repofetch
```
